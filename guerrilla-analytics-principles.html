3 Guerrilla Analytics Principles

Introductie

Guerrilla Analytics Principles zijn richtlijnen die helpen om data snel en flexibel te analyseren en de workflow te verbeteren. In deze hoofdstuk gebruiken we het Guerrilla Analytics-framework, zoals uitgelegd door Enda Ridge. Dit framework biedt een praktische en hands-on benadering voor het opzetten van een datamanagementproces dat zowel individueel als in teamverband effectief is. Hieronder wordt een voorbeeld gegeven van de principe met DAUR-II assignments:

library(knitr)

# Showing framework of RNA-sequence 
fs::dir_tree(here::here("daur2/RNAseq"))
# Showing framework of Metagenomics 
fs::dir_tree(here::here("daur2/Metagenomics"))
# Showing framework of ONECUT1 
fs::dir_tree(here::here("daur2/Eindopdracht"))

Figure 3.2: Folder tree Metagenomics. De folder tree van Metagenomics uit DAUR2 is gebaseerd op de Guerrilla Analytics Principle.



Figure 3.3: Folder tree ONECUT1. De folder tree van ONECUT1 uit de eindopdracht van DAUR2 is gebaseerd op de Guerrilla Analytics Principle.
